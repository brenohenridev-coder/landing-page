FAQ

PERGUNTAS
FREQUENTES

Preciso instalar alguma coisa?+
Não. O sistema funciona direto no navegador do celular ou computador. Basta acessar o link e fazer login.
Quantos professores podem usar?+
O plano inclui múltiplos professores. Cada um tem seu próprio login e senha. Basta nos informar os dados de acesso.
Os dados ficam salvos mesmo sem internet?+
Os dados ficam salvos na nuvem 24 horas por dia. Você precisa de internet para acessar, mas nunca perde os dados.
Posso cancelar quando quiser?+
Sim. Não há fidelidade nem multa. Cancele quando quiser pelo WhatsApp.
Como funciona o suporte?+
Suporte direto pelo WhatsApp. Respondemos rapidamente para resolver qualquer dúvida ou problema.
